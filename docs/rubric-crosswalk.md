# ✅ Rubric Crosswalk (Targets "Excellent")

**Design (113–125):**  
- Architecture, sequence diagrams, wireframe included in `docs/design.md`.  
- Clear rationale and direct mapping to modules.

**Explanation/Documentation (90–100):**  
- `README.md` has purpose, setup, usage, API and testing instructions.  
- Core logic (chunking, retrieval, summarization) explained.

**Applied Project Evaluation (275):**  
- Code reflects design: modules correspond 1:1 with described components.  
- Code runs: `uvicorn app.main:app` launches a working app; tests included.
